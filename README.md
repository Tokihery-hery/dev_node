# node_graphQL

### Installation npm
- Cloner repository

``git clone https://github.com/Tokihery-hery/node_graphQL.git``

Et run npm install

``npm install``

## Démarrage


- Lancez la serveur node 


``npm run start``
 
Et serveur in localhost:4000/graphql
